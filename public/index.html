Conexão com o Bling
O token do Bling fica armazenado com segurança no servidor (Render), não aparece nesta tela para os funcionários.

Para atualizar o token, acesse o painel do Render → seu serviço → Environment → variável BLING_TOKEN.
⚠ O token OAuth do Bling expira a cada 6 horas. Quando o "Buscar no Bling" der erro, basta atualizar a variável no Render com um token novo.
